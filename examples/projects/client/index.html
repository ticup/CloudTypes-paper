CloudTypes Example: Projects

Project Manager to demonstrate Entities and weak Entities.
Consists of Members, Projects and Tasks, where a Task can be given to a certain Member for a certain Project and for which priority can be voted.
All entities can be edited by clicking them! Track/Adjust connectivity using the online/offline buttons.
